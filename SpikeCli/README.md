# Welcome to SpikeCli 🤗

## About
🚀 SpikeCli was created to help me (mikkelgr) **build application interfaces as fast as possible**. To this day, spikeCli is the fastest way I've discovered. 

🐌 Creating CLIs was already the fastest kind of interface to create and to user,
but CLI libraries will usually have you create a class per command and a class per the arguemtns/options for each command.

🛠️ **SpikeCli uses the builder pattern** (some call it fluent syntax) to specify the CLI. 
For me, this I much faster.
The interface of **a 20-command app can be specified on one screen/20 lines**, as opposed to 20*2 classes. 

> **The tradeoff:** On startup, the app will build the CLI specification in memory. This is not free, but negligible for most use-cases.


📦 **I've nuget-package-up the tool and share it with anyone who find it useful. Cheers!**
